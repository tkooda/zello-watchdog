# zello-watchdog
A simple background Android app to ensure Zello app is always running

# Install
You can install an [unpacked zip](https://github.com/tkooda/zello-watchdog/releases/download/v1.4/z-watchdog_1_4.apk), but Chrome will warn you about it every time you start the browser.

It's recommended you install the extension directly from the [Google Web Store](https://chrome.google.com/webstore/detail/jekeilmcchdcmodcmknhdehocdgdkcbl) instead (to avoid the warning, and obtain updates).
